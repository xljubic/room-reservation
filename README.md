# Room Reservation (Backend)

Backend REST API for room/sala reservations (FON course project).

## Tech stack
- Java 17+
- Spring Boot (Spring Web, Spring Data JPA)
- Maven
- (Planned) Liquibase for DB migrations
- (Planned) Swagger UI / OpenAPI for API documentation

## Project structure
- `src/main/java` – application source code
- `src/main/resources` – application configuration

## Requirements
- Java 17 or newer (Spring Boot 4.x)  
- Maven (or use Maven Wrapper: `mvnw` / `mvnw.cmd`)

## How to run
From the project root:

### Windows (PowerShell / CMD)
```bash
mvnw.cmd spring-boot:run
